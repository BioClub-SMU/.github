## 🌟 关于我们

**身心医学研究会 · 生信小组** 是身心医学研究会旗下的技术实践小组，专注于：

- 🧬 **生物信息学分析**（如RNA-seq、单细胞、甲基化、CNV、miRNA等）
- 🧠 **机器学习建模**（支持疾病预测、多组学融合、图神经网络等）
- 📊 **定量生物学研究**（建模、参数估计、动力学模拟）
- 📈 **流行病学统计分析**（Cox模型、因果推断、政策评估、RCT分析）

我们致力于在生命医学的真实问题中，推广开放协作与可重复研究。
